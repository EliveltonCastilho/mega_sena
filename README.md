# Mega Estrategista Pro

Aplicativo web para análise estatística de jogos da Mega-Sena, utilizando dados históricos reais para gerar recomendações baseadas em ciência de dados.

## Funcionalidades

- Análise estatística baseada em histórico real de concursos
- Geração automática de números com filtros inteligentes
- Seleção manual de números com feedback em tempo real
- Visualização de distribuição espacial em quadrantes
- Análise de paridade (pares/ímpares)
- Histograma real da "curva de sino" de somas históricas
- Comparação com último sorteio real

## Tecnologias

- React 18
- Vite
- Lucide React (ícones)
- Tailwind CSS

## Instalação

```bash
npm install
```

## Desenvolvimento

```bash
npm run dev
```

## Build

```bash
npm run build
```
